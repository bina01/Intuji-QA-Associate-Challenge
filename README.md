# Setup steps:
 - git clone https://github.com/bina01/Intuji-QA-Associate-Challenge.git 
 - npm install 

# Run tests
- npx cypress open (to open cypress test runner in browser)
- npx cypress run (to run in terminal)

- Cypress
- faker.js
- Cypress fixtures
- Page Object Model(POM)
